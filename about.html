About Stellaris

Stellaris is an educational static website designed to showcase the rich history and exciting future of space exploration.

Project Details

Stack: HTML5, CSS3 (Vanilla), Vanilla JavaScript

Design System: Dark Mode Native, Glassmorphism, CSS Mesh Gradients

Typography: 'Outfit' and 'Inter' from Google Fonts

Created as a premium demonstration of modern web design techniques without the use of heavy frameworks or external imagery.
